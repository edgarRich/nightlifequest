# NightlifeQuest

NightlifeQuest is a gamified nightlife engagement platform for clubs and DJs.

## Features
- QR-based club check-in
- Live leaderboards
- Club challenges & rewards
- Admin dashboard for venues

## Tech Stack
- Frontend: Web (Netlify)
- Backend: Supabase (Auth, Postgres, Realtime)
- Mobile (Planned): Flutter

## Setup
1. Clone the repo
2. Create `.env` file
3. Add Supabase credentials
4. Run locally
